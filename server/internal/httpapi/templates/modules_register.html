Register Module

← Modules

Register a new module by its firmware-configured ID, or commission a module that has already been discovered (auto-created on first heartbeat). The module ID must match CONFIG_PORTUNUS_MODULE_ID in the firmware Kconfig.

Must match the ID set in the module firmware exactly.

You can assign or reassign the door later from the module detail page.
